# Hi, I'm Lauri

I'm a fourth-year Embedded & Smart Systems student with a strong interest in embedded software and firmware development.

What I enjoy most about embedded engineering is writing software that interacts with the physical world. Seeing code come alive on real hardware is what motivated me to pursue embedded systems in the first place.

While I'm particularly interested in medical devices and healthcare technology, I'm always looking for opportunities to learn and work on interesting embedded projects across different industries.

## Current Project

I'm currently developing the pulse oximeter subsystem for **SAMD (Sleep Apnea Monitoring Device)**, a university project focused on monitoring sleep apnea. My work involves developing firmware for an ESP32-C6 using the MAX30102 optical sensor together with FreeRTOS.
